# ArtContent 插件说明

- 功能分类：内容资源
- 主要功能：
  - 作为内容（资源）型插件，集中存放项目通用的美术资源（材质、纹理、模型、音频、数据资产等），便于模块化分发与复用。
  - 不包含或极少包含运行时代码，主要通过内容引用被项目其余部分使用。

- 关键模块与类/对象：
  - 资源与数据资产：该插件的核心是各类 UAsset（如材质实例、纹理、静态/骨骼网格、数据表等）。

- 典型交互关系：
  - 关卡/蓝图/其他插件 -> 引用 ArtContent 中的资源，用于表现、美术风格统一与数据驱动。

- 参考：
  - 插件清单：ArtContent.uplugin
  - 资源路径：Plugins/ArtContent/Content（及子目录）
