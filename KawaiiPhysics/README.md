# KawaiiPhysics 插件说明

- 功能分类：物理模拟
- 主要功能：
  - 实现角色附件（头发、饰品、布料等）的次级物理模拟（摆动/抖动），以较低的成本获得自然的跟随与阻尼效果。

- 关键模块与类/对象：
  - KawaiiPhysics 提供的物理节点/组件（如骨骼控制器）：用于在动画蓝图中对指定骨骼链施加物理模拟与约束。
  - 参数与约束：通过插件提供的参数（阻尼、弹性、重力、限制等）实现风格化与稳定性调优。

- 典型交互关系：
  - 动画蓝图 -> KawaiiPhysics 节点/组件：在动画图中插入节点驱动骨骼链，实现实时物理响应。

- 参考：
  - 插件清单：KawaiiPhysics.uplugin
  - 源码路径：Plugins/KawaiiPhysics/Source/KawaiiPhysics
  - 使用方式：在角色动画蓝图中添加相应的 KawaiiPhysics 节点，配置受影响的骨骼与参数。
