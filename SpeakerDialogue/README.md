# SpeakerDialogue 插件说明

- 功能分类：音频 / 语音 / 对白
- 主要功能：
  - 提供对白/播报相关的辅助逻辑，集中管理“说话者”与对白播放控制。
  - 便于在关卡/剧情中以统一接口触发对白、控制播放状态与回调。

- 关键模块与类/对象：
  - 模块入口（SpeakerDialogue 模块）：负责模块初始化/反初始化、注册必要的类型与资源。
  - 对白控制组件/工具：用于在 Actor 上驱动对白（开始/暂停/停止）、与字幕或动画联动（具体类见源码）。

- 典型交互关系：
  - 游戏逻辑/蓝图 -> 对白控制组件：触发对白、监听结束回调。

- 参考：
  - 插件清单：SpeakerDialogue.uplugin
  - 源码路径：Plugins/SpeakerDialogue/Source/SpeakerDialogue
